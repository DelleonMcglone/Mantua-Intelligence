# Mantua-Intelligence

Mantua.AI is an agent-driven liquidity protocol that lets anyone interact with crypto markets by simply typing what they want. It combines a custom suite of Uniswap v4 hooks, autonomous AI agents, and real-time blockchain access into a single programmable liquidity layer.

## Supported Network

Mantua.AI is being built on **Base**.

| Network      | Chain ID | Status      | Block Explorer              |
