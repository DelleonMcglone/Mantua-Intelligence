```
**Source:** <repo@commit, file:line>
**Severity:** Critical | High | Medium | Low | Info

**Reporter:** <name / tool>
**Reported:** <YYYY-MM-DD>

### Description
<what's wrong>

### Reproducer
<how to trigger>

### Recommendation
<proposed fix>
```
